# ZhuHuSpider
ZhuHuSpider
批量下载知乎上的图片

使用第三方库：
requests, BeautifulSoup

使用方法：
img_url：下载知乎图片的问题链接
path：下载保存的路径
将两个变量改为用户自定义即可
